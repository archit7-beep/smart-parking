# Smart Parking System

A smart parking management system built using:
- Flask (Python)
- Firebase Firestore
- Google Sheets
- Looker Studio

## Features
- Web-based parking entry form
- Stores data in Firebase
- Syncs data to Google Sheets
- Real-time analytics using Looker Studio

## Tech Stack
- Frontend: HTML (Flask templates)
- Backend: Python + Flask
- Database: Firebase Firestore
- Analytics: Looker Studio
